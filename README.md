Community Connect is a production-grade full-stack web application designed to digitally connect volunteers with community organizations for efficient event coordination and management. The platform streamlines volunteer registration, event management and communication through a secure and scalable architecture.

# Features

1. Volunteer & Organization Management
Role-based system allowing volunteers and organizations to manage events and participation effectively.

2. Secure Authentication

    -Traditional email/password login

    -Google OAuth 2.0 integration

    -Session management using Flask-Login

    -Role-based access control (RBAC)

3. Event Management
Organizations can create, update, and manage events, while volunteers can register or cancel participation.

4. AJAX-Powered Interactions
Seamless volunteer signups, cancellations, and admin operations without page reloads.

5. Email Notifications
SMTP-based transactional emails for confirmations and updates.

6. External API Integration
Public weather and geolocation APIs enhance event information and context.
